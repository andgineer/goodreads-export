API Overview

Modules

  • author_file: Author's file.
  • authored_file: Library object with author.
  • book_file: Book's file.
  • clean_file_name: Make file name safe for cloud disks.
  • data_file: Object stored in the file.
  • goodreads_book: Read goodreads export.
  • library: Library of ooks.
  • log: Logger.
  • main: Create md-files from https://www.goodreads.com/ CSV export.
  • series_file: Series file.
  • stat: Loading statistics.
  • templates: Templates for the Goodreads Exporter.
  • version

  • goodreads_book.GoodreadsBooks: List of books from goodreads export.
  • library.Library: Books and authors.
  • log.Log: Logger or progress bar.
  • series_file.SeriesFile: Series' file.

  • templates.BookTemplate: Book template.
